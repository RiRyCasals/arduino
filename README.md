## ボタンの仕様

左から C, D, E, F, G, A, B （ドレミファソラシド）

## ジョイスティックの仕様

- 上に倒す：半音上げる
- 下に倒す：半音下げる
- 右に倒す：1オクターブ上げる
- 左に倒す：1オクターブ下げる

## ディスプレイの仕様

音名（アルファベット）とその周波数を表示
